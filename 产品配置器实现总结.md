# 产品配置器实现总结

## 🎯 项目概述

基于RuoYi-Vue3框架和现有超级BOM规则配置系统，成功开发了一套完整的**用户选型配置器**（Product Configurator）前端解决方案。该配置器符合主流CPQ系统设计模式，实现了与超级BOM规则的深度集成和动态交互。

## 🎓 核心算法解析

### 1. 规则条件评估算法

```javascript
/**
 * 递归评估嵌套条件
 * 时间复杂度: O(n) - n为条件数量
 * 空间复杂度: O(d) - d为嵌套深度
 */
evaluateConditions(conditions) {
  let result = null
  
  for (let i = 0; i < conditions.length; i++) {
    const condition = conditions[i]
    
    // 递归处理复杂条件
    const conditionResult = condition.configType === 'complex'
      ? this.evaluateConditions(condition.children)
      : this.evaluateSingleCondition(condition)
    
    if (i === 0) {
      result = conditionResult
    } else {
      // 逻辑运算符连接
      const logic = condition.logic || 'AND'
      result = logic === 'AND' 
        ? result && conditionResult 
        : result || conditionResult
    }
  }
  
  return result
}
```

**算法特点：**
- 支持任意深度嵌套
- 短路求值优化
- 支持AND/OR逻辑组合

### 2. BOM树扁平化算法

```javascript
/**
 * 将树形结构扁平化为Map
 * 时间复杂度: O(n) - n为节点总数
 * 空间复杂度: O(n)
 */
buildNodesMap(nodes) {
  nodes.forEach(node => {
    // 初始化节点状态
    node.status = 'unconfigured'
    node.isVisible = true
    // ...
    
    // 存入Map (O(1)查询)
    this.nodesMap.set(node.bomStructureId, node)
    
    // 递归处理子节点
    if (node.children && node.children.length > 0) {
      this.buildNodesMap(node.children)
    }
  })
}
```

**优势：**
- O(1)时间复杂度查询节点
- 便于节点状态更新
- 支持树形和扁平视图切换

### 3. 价格计算算法

```javascript
/**
 * 计算总价（支持自定义表达式）
 * 时间复杂度: O(n) - n为已配置节点数
 */
async calculateTotalPrice() {
  let totalPrice = 0
  const breakdown = []

  this.nodesMap.forEach((node, nodeId) => {
    if (node.status === 'configured' && node.isVisible) {
      let nodePrice = node.price || 0
      
      // 自定义计算规则
      if (node.costCalculationRule) {
        const context = RuleEngine.getContext()
        const func = new Function(
          ...Object.keys(context),
          `return ${node.costCalculationRule}`
        )
        nodePrice = func(...Object.values(context))
      }
      
      const totalNodePrice = nodePrice * node.quantity
      totalPrice += totalNodePrice
      
      breakdown.push({
        nodeId,
        nodeName: node.nodeName,
        quantity: node.quantity,
        unitPrice: nodePrice,
        totalPrice: totalNodePrice
      })
    }
  })

  this.pricing = { totalPrice, breakdown }
}
```

**特点：**
- 支持基础价格 × 数量
- 支持产品价格覆盖
- 支持自定义计算表达式
- 提供详细价格明细
